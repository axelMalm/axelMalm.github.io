Integritetspolicy för Interactive Drawing Senast uppdaterad: [datum] Interactive Drawing värnar om din integritet. Denna policy beskriver hur vi samlar in, använder och skyddar information när du använder vår mobilapplikation. 1. Insamling av information Interactive Drawing samlar inte in personligt identifierbar information från användare. Appen kan dock automatiskt samla in viss icke-personlig information för att förbättra funktionaliteten, såsom: Enhetstyp och operativsystem Appanvändning (t.ex. funktioner som används) Felrapporter och prestandadata Denna information kan samlas in via tredjepartstjänster som används för appens drift och stabilitet. 2. Användning av information Den information som samlas in används enbart för att: Förbättra appens funktion och användarupplevelse Identifiera och åtgärda tekniska problem Säkerställa appens stabilitet och säkerhet Vi säljer, handlar eller delar inte användarinformation med tredje part för marknadsföringsändamål. 3. Barns integritet Interactive Drawing riktar sig inte specifikt till barn under 13 år. Vi samlar inte medvetet in personuppgifter från barn. Om du anser att ett barn har lämnat personlig information, vänligen kontakta oss så raderar vi den omgående. 4. Datasäkerhet Vi vidtar rimliga tekniska och organisatoriska åtgärder för att skydda information mot obehörig åtkomst, ändring eller radering. 5. Tredjepartstjänster Appen kan använda tredjepartsbibliotek eller tjänster som kan samla in anonym användningsdata i enlighet med deras egna integritetspolicyer. 6. Ändringar i denna policy Vi kan uppdatera denna integritetspolicy vid behov. Ändringar publiceras på denna sida med nytt uppdateringsdatum. 7.
